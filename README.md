# Overview
A simple health tracking app.

The app provides the following facilites
- Date
- Time
- Systolic
- Diastolic
- Heart Rate
- Commnets

# Why CheckHealth
- Attractive UI
- Easy to add
- Easy to modify
- Easy to delete
